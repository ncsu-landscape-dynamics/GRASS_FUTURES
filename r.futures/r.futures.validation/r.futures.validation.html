DESCRIPTION

Module r.futures.gridvalidation is a support tool for validation of FUTURES simulation results. It computes allocation disagreement, quantity disagreement, Cohen's kappa and Kappa simulation. Input raster simulated should be an output from r.futures.pga module. Input raster reference is a reference raster with 0 as undeveloped and 1 as developed pixels. Input raster original represents the initial conditions (0 as undeveloped and 1 as developed) and is needed only for Kappa simulation.

EXAMPLES

Validate FUTURES output by computing quantity and allocation disagreement: 
r.futures.validation simulated=simulated_2016 reference=reference_2016 original=orig_2001 format=
